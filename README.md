# Проект: «Изучение и анализ возможностей применения технологии Lightning Network»

Главная цель проекта: изучить технологию Lightning Network, найти и проанализировать доступную в сети информацию по теме криптовалют и криптокошельков,протестировать различные веб и мобильные приложения, которые основаны на технологии LN. 

Преимущество данного проекта: изучение профессии аналитик, тестирование приложений с целью выявления преимуществ,недостатков,а также неточностей в работе, написание отчётов по проведенной работе.
